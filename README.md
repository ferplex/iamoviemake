# 📚 Mangaia Pro

> Tu plataforma para descubrir y leer manga online.

![Versión](https://img.shields.io/badge/Versión-1.0.0-blue)
![Estado](https://img.shields.io/badge/Estado-Activo-green)

**Mangaia Pro** es una plataforma web que te permite explorar un amplio catálogo de manga, seguir tus series favoritas y disfrutar de una experiencia de lectura cómoda y sin distracciones.

---

## ✨ Características

- 📖 **Lector Integrado:** Interfaz limpia y optimizada para lectura continua.
- 🔍 **Búsqueda y Filtros:** Encuentra manga por título, género, autor o estado.
- 📊 **Seguimiento Personal:** Marca capítulos como leídos y organiza tu lista de pendientes.
- 🌙 **Modo Oscuro/Claro:** Adapta la interfaz a tu preferencia.
- 📱 **Diseño Responsivo:** Funciona perfectamente en móviles, tablets y escritorio.
- ⚡ **Carga Rápida:** Optimizado para una experiencia fluida.

## 🚀 Instalación Local

```bash
# Clona el repositorio
git clone https://github.com/[tu-usuario]/mangaia-pro.git
cd mangaia-pro

# Instala dependencias
npm install

# Configura variables de entorno
cp .env.example .env
# Edita el archivo .env con tus configuraciones

# Inicia el servidor
npm run dev
